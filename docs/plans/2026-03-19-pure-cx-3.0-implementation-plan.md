# Pure CX 3.0 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 `cx-workflow` 从当前的 `2.0 + 本地演化` 状态收敛为纯 `cx 3.0` 插件内核，让所有项目都以项目级 `.claude/cx` 为唯一运行时真相，并把轻量体验内化到现有 `cx` 命令中。

**Architecture:** 本次改造分三层推进。先建立 `3.0` 协议与校验基座，再重构 `cx-init + hooks + planning/execution` 的运行主链，最后统一 README、workflow-guide、插件元数据等对外叙事。所有改动只在插件仓库完成，不处理历史 `tc` 兼容层。

**Tech Stack:** Markdown, JSON Schema, Bash hooks, Claude Code skill docs, Git, jq, rg, bash.
